Для запуска потребуется Java 1.7 или выше

Запуск
  Windows: main.bat
  Linux: sh main.sh
  Или на любой платформе: java -jar dh.jar

Управление:
  NUM+ / NUM- : изменение масштаба
  1 2 3 4 5 6 : изменение скорости времени на глобальной карте
  spacebar : пауза
  
В игре есть две части:
	глобальная, когда мы смотрим на карту, строим гильдии и т.д.
	боевая, когда мы выезжаем управляем отрядом

Чтобы попасть в боевую часть надо:
1. дождаться повления врага (можно прокрутить время выставив скорость на 5)
2. нажать на гильдию на карте (она в виде красного флага, перейти к ней можно нажав на название в списке справа сверху)
3. нажать на появившуюся стрелку над гильжией
4. выбрать место противника (красная пентаграмма на карте)
